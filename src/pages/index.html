常见问题

解答您可能遇到的问题

Cool Request 支持 IntelliJ IDEA 2020.1 及以上版本，包括 Community 和 Ultimate 版本。同时也支持 WebStorm、PyCharm、GoLand 等基于 IntelliJ 平台的 IDE。

您可以通过以下两种方式安装 Cool Request：

  1. 在 IDEA 中，依次点击 File > Settings > Plugins，搜索 "Cool Request" 并安装。
  2. 从我们的官网下载插件包（.zip 文件），然后在 IDEA 中，依次点击 File > Settings > Plugins > ⚙️ > Install Plugin from Disk，选择下载的插件包安装。

是的，Cool Request 是完全免费的开源项目，您可以在 GitHub 上查看源代码并参与贡献。我们欢迎社区的反馈和建议，以不断改进插件功能。

要调用 Spring 容器内的方法，您需要：

  1. 确保您的 SpringBoot 应用已启动并运行。
  2. 在 Cool Request 工具窗口中，切换到 "Spring" 标签页。
  3. 在左侧树形视图中，浏览并选择要调用的 Bean。
  4. 在右侧面板中，选择要调用的方法，填写参数值。
  5. 点击 "执行" 按钮调用方法，查看返回结果。

导出 API 文档的步骤如下：

  1. 在 Cool Request 工具窗口中，点击右上角的 "导出" 按钮。
  2. 在弹出的对话框中，选择要导出的 API 和导出格式（如 HTML、Markdown、Word 等）。
  3. 选择保存路径，点击 "导出" 按钮。
  4. 文档生成完成后，会自动打开文档或提示导出成功。
